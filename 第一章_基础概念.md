# 会计学第一章：基础概念

---

## 一、会计的定义

**会计**是对主体（企业、组织）的经济活动进行核算和监督的信息系统。

| 要素 | 说明 |
|------|------|
| 核算 | 确认、计量、记录、报告 |
| 监督 | 审核、控制、分析 |

**会计的目标**：为信息使用者提供决策有用的财务信息。

---

## 二、会计的基本假设

| 假设 | 含义 | 实际应用 |
|------|------|----------|
| **会计主体** | 会计核算的范围是企业本身 | 企业与个人财务分开 |
| **持续经营** | 企业将持续存在下去 | 资产按历史成本计价 |
| **会计分期** | 将企业经营划分时间段 | 月报、季报、年报 |
| **货币计量** | 以货币为主要计量单位 | 统一衡量业务 |

---

## 三、会计六要素

### 3.1 资产
- **定义**：企业过去的交易形成的，由企业控制的，预期带来经济利益的资源
- **特点**：拥有或控制、能以货币计量、预期带来收益
- **常见科目**：银行存款、应收账款、固定资产、无形资产

### 3.2 负债
- **定义**：企业过去的交易形成的，预期导致经济利益流出的现时义务
- **特点**：现时义务、经济利益很可能流出、金额可靠
- **常见科目**：短期借款、应付账款、应付票据

### 3.3 所有者权益
- **定义**：资产扣除负债后由所有者享有的剩余权益
- **组成**：实收资本、资本公积、盈余公积、未分配利润

### 3.4 收入
- **定义**：企业在日常活动中形成的、导致所有者权益增加的、与所有者投入资本无关的经济利益总流入
- **特点**：日常活动产生、导致权益增加、与投入资本无关

### 3.5 费用
- **定义**：企业在日常活动中发生的、导致所有者权益减少的、与向所有者分配利润无关的经济利益总流出
- **特点**：日常活动产生、导致权益减少、与分配无关

### 3.6 利润
- **定义**：企业在一定会计期间的经营成果
- **公式**：利润 = 收入 - 费用

---

## 四、会计等式

### 4.1 基本等式
```
资产 = 负债 + 所有者权益
```

这是会计最核心的等式，所有经济业务都围绕这个等式展开。

### 4.2 扩展等式
```
资产 = 负债 + 所有者权益 + (收入 - 费用)
```

或：
```
资产 + 费用 = 负债 + 所有者权益 + 收入
```

---

## 五、经济业务对等式的影响

| 业务类型 | 举例 | 影响 |
|----------|------|------|
| **同增** | 收到投资 | 资产↑ + 所有者权益↑ |
| **同减** | 归还借款 | 资产↓ + 负债↓ |
| **有增有减（等式一边）** | 用银行存款还债 | 资产↓ + 负债↓ |
| **有增有减（等式一边）** | 用利润购买设备 | 资产↑ + 资产↓（同增同减） |

---

## 六、借贷记账法

**核心规则**：
- **借方**：记录资产增加、费用增加、负债减少、所有者权益减少、收入减少
- **贷方**：记录负债增加、所有者权益增加、收入增加、资产减少、费用减少

**口诀**：有借必有贷，借贷必相等
