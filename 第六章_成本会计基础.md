## 一、成本会计概述

### 1.1 成本会计的定义

**成本会计**是会计学科的重要分支，专注于产品或服务成本的核算与分析。

| 概念 | 定义 |
|------|------|
| 成本 | 企业为生产产品或提供服务而发生的各项耗费 |
| 费用 | 企业在日常活动中发生的、会导致所有者权益减少的经济利益流出 |
| 损失 | 企业非日常活动发生的经济利益流出 |

### 1.2 成本会计的作用

1. **定价决策** - 提供成本信息支持定价
2. **成本控制** - 监控和分析成本构成
3. **经营决策** - 提供经营效益分析
4. **绩效考核** - 评估各部门成本绩效

## 二、成本分类

### 2.1 按计入产品方式

| 类型 | 定义 | 举例 |
|------|------|------|
| **直接成本** | 直接计入产品成本 | 原材料、直接人工 |
| **间接成本** | 需要分配计入产品成本 | 制造费用、水电费 |

### 2.2 按性态分类

| 类型 | 定义 | 特性 |
|------|------|------|
| **固定成本** | 不随产量变动而变动 | 厂房折旧、管理人员工资 |
| **变动成本** | 随产量成正比例变动 | 直接材料、计件工资 |
| **混合成本** | 同时具有固定和变动特性 | 维修费、电话费 |

### 2.3 成本结构图

```
总成本
  │
  ├── 固定成本（稳定）
  │     ├── 厂房折旧
  │     ├── 管理人员工资
  │     └── 租金
  │
  └── 变动成本（随产量变化）
        ├── 直接材料
        ├── 直接人工
        └── 计件工资
```

## 三、成本核算流程

### 3.1 基本步骤

```
确定成本计算对象 → 归集费用 → 分配费用 → 计算产品成本
```

### 3.2 成本核算主要科目

| 科目代码 | 科目名称 | 说明 |
|----------|----------|------|
| 5001 | 生产成本 | 核算产品生产成本 |
| 5101 | 制造费用 | 核算生产车间组织生产发生的费用 |
| 4101 | 生产成本 | 主营业务成本 |

### 3.3 成本核算流程图

```
领料单 → 原材料分配 → 基本生产成本
                    ↓
工资表 → 直接人工分配 → 在产品
                    ↓
制造费用汇总 → 制造费用分配 → 完工产品成本
```

## 四、制造成本计算方法

### 4.1 品种法（简单法）

**适用**：大量大批单步骤生产（如发电、采掘）

```
特点：
- 以产品品种为成本计算对象
- 月末一般无在产品或在产品很少
- 成本计算期与会计期间一致
```

### 4.2 分批法（订单法）

**适用**：单件小批生产（如船舶、飞机）

```
特点：
- 以批次为成本计算对象
- 成本计算期不固定（与生产周期一致）
- 一般无在产品或在产品跨期
```

### 4.3 分步法

**适用**：大量大批多步骤生产（如纺织、冶金）

| 方法 | 特点 | 适用 |
|------|------|------|
| 逐步结转分步法 | 逐步结转半成品成本 | 需要考核半成品成本 |
| 平行结转分步法 | 各步骤只计算应计入最终产品的份额 | 不需要考核半成品成本 |

## 五、MySQL成本核算表设计

### 5.1 产品成本表

```sql
CREATE TABLE product_costs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    product_code VARCHAR(20) NOT NULL COMMENT '产品编码',
    cost_period DATE NOT NULL COMMENT '成本期间',
    direct_material DECIMAL(12,2) DEFAULT 0 COMMENT '直接材料',
    direct_labor DECIMAL(12,2) DEFAULT 0 COMMENT '直接人工',
    manufacturing_overhead DECIMAL(12,2) DEFAULT 0 COMMENT '制造费用',
    total_cost DECIMAL(12,2) DEFAULT 0 COMMENT '总成本',
    unit_cost DECIMAL(12,2) DEFAULT 0 COMMENT '单位成本',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (product_code) REFERENCES accounting_subjects(subject_code)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='产品成本表';
```

### 5.2 成本明细表

```sql
CREATE TABLE cost_details (
    id INT AUTO_INCREMENT PRIMARY KEY,
    product_code VARCHAR(20) NOT NULL COMMENT '产品编码',
    cost_type ENUM('direct_material','direct_labor','overhead') NOT NULL COMMENT '成本类型',
    amount DECIMAL(12,2) NOT NULL COMMENT '金额',
    quantity DECIMAL(12,2) DEFAULT 0 COMMENT '数量',
    unit_price DECIMAL(12,2) DEFAULT 0 COMMENT '单价',
    period_start DATE NOT NULL COMMENT '期间开始',
    period_end DATE NOT NULL COMMENT '期间结束',
    FOREIGN KEY (product_code) REFERENCES accounting_subjects(subject_code)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='成本明细表';
```

### 5.3 成本分析视图

```sql
CREATE VIEW cost_analysis AS
SELECT 
    product_code,
    cost_period,
    direct_material,
    direct_labor,
    manufacturing_overhead,
    total_cost,
    unit_cost,
    -- 计算各成本占比
    ROUND(direct_material / total_cost * 100, 2) AS material_ratio,
    ROUND(direct_labor / total_cost * 100, 2) AS labor_ratio,
    ROUND(manufacturing_overhead / total_cost * 100, 2) AS overhead_ratio
FROM product_costs
WHERE total_cost > 0;
```

## 六、成本计算实例

### 6.1 资料

某企业生产A产品，本月发生以下费用：

| 项目 | 金额（元） |
|------|------------|
| 直接材料 | 200,000 |
| 直接人工 | 80,000 |
| 制造费用 | 40,000 |

本月完工产品100件，在产品50件（完工率50%）

### 6.2 计算过程

**1. 计算约当产量**

```
完工产品：100件
在产品约当产量：50件 × 50% = 25件
约当总产量：100 + 25 = 125件
```

**2. 计算单位成本**

```
直接材料分配率 = 200,000 / 125 = 1,600元/件
直接人工分配率 = 80,000 / 125 = 640元/件
制造费用分配率 = 40,000 / 125 = 320元/件

单位成本 = 1,600 + 640 + 320 = 2,560元/件
```

**3. 成本计算结果**

| 项目 | 总成本 | 单位成本 |
|------|--------|----------|
| 直接材料 | 160,000 | 1,600 |
| 直接人工 | 64,000 | 640 |
| 制造费用 | 32,000 | 320 |
| **完工产品成本** | **256,000** | **2,560** |

## 七、成本控制基础

### 7.1 标准成本法

| 概念 | 定义 |
|------|------|
| 标准成本 | 预先确定的正常成本 |
| 成本差异 | 实际成本与标准成本的差额 |
| 差异分析 | 分析差异产生的原因 |

### 7.2 成本差异计算

```
成本差异 = 实际成本 - 标准成本

差异分析：
- 节约差异（负差）：实际 < 标准
- 超支差异（正差）：实际 > 标准
```

### 7.3 成本控制方法

| 方法 | 说明 |
|------|------|
| 预算控制 | 按预算标准控制成本 |
| 目标控制 | 设定目标成本进行控制 |
| 责任控制 | 按责任中心进行控制 |
