## 一、会计报表概述

### 1.1 报表的定义

**会计报表**是会计期末对企业财务状况、经营成果的书面报告。

### 1.2 主要报表

| 报表 | 反映内容 | 时间 |
|------|----------|------|
| 资产负债表 | 某一时点财务状况 | 期末 |
| 利润表 | 某一时期经营成果 | 期间 |
| 现金流量表 | 现金流入流出 | 期间 |

---

## 二、资产负债表

### 2.1 定义

反映企业在某一特定日期的财务状况的报表。

**格式：**
```
资产 = 负债 + 所有者权益
```

### 2.2 资产负债表格式

| 资产 | 金额 | 负债和所有者权益 | 金额 |
|------|------|------------------|------|
| 流动资产 |  | 流动负债 |  |
| 货币资金 | 113,000 | 应付账款 | 30,000 |
| 应收账款 | 50,000 | 短期借款 | 20,000 |
| 存货 | 80,000 | 负债合计 | 50,000 |
| 非流动资产 |  | 所有者权益 |  |
| 固定资产 | 200,000 | 实收资本 | 350,000 |
| 累计折旧 | -30,000 | 未分配利润 | 13,000 |
| 资产合计 | 413,000 | 负债和权益合计 | 413,000 |

### 2.3 编制公式

```
资产 = 流动资产 + 非流动资产
负债 = 流动负债 + 非流动负债
所有者权益 = 实收资本 + 资本公积 + 盈余公积 + 未分配利润
```

---

## 三、利润表

### 3.1 定义

反映企业在一定会计期间的经营成果的报表。

### 3.2 利润表格式

| 项目 | 本期金额 |
|------|----------|
| 一、营业收入 | 500,000 |
| 减：营业成本 | 300,000 |
| 二、营业利润 | 200,000 |
| 减：管理费用 | 50,000 |
| 减：财务费用 | 10,000 |
| 三、利润总额 | 140,000 |
| 减：所得税费用 | 35,000 |
| 四、净利润 | 105,000 |

### 3.3 计算公式

```
营业利润 = 营业收入 - 营业成本 - 费用
利润总额 = 营业利润 + 营业外收入 - 营业外支出
净利润 = 利润总额 - 所得税费用
```

## 四、现金流量表（简化）

### 4.1 定义

反映企业现金流入和流出的报表。

### 4.2 现金来源分类

| 类别 | 内容 |
|------|------|
| 经营活动 | 销售商品、提供劳务 |
| 投资活动 | 购买设备、投资收益 |
| 筹资活动 | 借款、发行股票 |

---

## 五、会计报表之间的关系

```
会计等式：资产 = 负债 + 所有者权益

        资产负债表
            ↓
    ┌──────────────┐
    ↓              ↓
 利润表        现金流量表
 (净利润)       (现金)
    ↓              ↓
    └──────────────┘
            ↓
    所有者权益变动
```

**关系：**
- 资产负债表是"照片"（某一时点）
- 利润表是"录像"（某一时期）
- 现金流量表追踪"真金白银"

## 六、MySQL报表设计

### 6.1 试算平衡表（验证）

```sql
-- 试算平衡表：检查借贷是否平衡
CREATE VIEW trial_balance AS
SELECT 
    subject_code,
    SUM(CASE WHEN direction='借' THEN amount ELSE 0 END) AS debit_total,
    SUM(CASE WHEN direction='贷' THEN amount ELSE 0 END) AS credit_total
FROM accounting_voucher_items
GROUP BY subject_code
HAVING debit_total != credit_total;
-- 无结果 = 平衡
```

### 6.2 利润表视图

```sql
CREATE VIEW income_statement AS
SELECT 
    '营业收入' AS item,
    SUM(CASE WHEN subject_code LIKE '4%' THEN amount ELSE 0 END) AS amount
FROM accounting_voucher_items
UNION ALL
SELECT 
    '管理费用',
    SUM(CASE WHEN subject_code = '6602' THEN amount ELSE 0 END)
FROM accounting_voucher_items;
```

### 6.3 资产负债表视图

```sql
CREATE VIEW balance_sheet AS
SELECT 
    subject_type,
    SUM(CASE WHEN direction='借' THEN amount ELSE -amount END) AS balance
FROM accounting_subjects s
JOIN accounting_voucher_items i ON s.subject_code = i.subject_code
GROUP BY subject_type;
```
